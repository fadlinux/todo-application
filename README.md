# Todo application
Demo Todo Tasks Application 

## Requirements 
- java 22.0.2 
- gradle
- mysql version 8
- Database migration tools with java spring-jpa hibernate

## Preparation
- create database name 'todo_db'
- import database.sql to MySQL server

- sh gradlew bootRun

- sh gradlew build
