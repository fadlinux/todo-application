# Todo application
Demo Todo Tasks Application 

## Requirements 
- java 22.0.2 
- gradle
- mysql version 8

## Preparation
- import database.sql to MySQL server

## How To Run
- sh gradlew bootRun

## How To Build
- sh gradlew build

## How to test API 
- import file Todo-Task-Application.postman_collection.json to Postman application
